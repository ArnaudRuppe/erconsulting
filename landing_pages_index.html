About This Project

This project demonstrates multiple landing page variations for ERConsulting, a specialized wine industry consulting firm. All variations maintain the same core content while differing in layout structure, visual hierarchy, component arrangement, and overall design approach.

Same Content

All variations present identical text content about hygiène, process, and matériel consulting services.

Different Layouts

Each variation features distinct section organization, visual hierarchy, and component arrangement approaches.

Consistent Branding

All pages use the same color scheme, typography, and tech stack for brand consistency.
